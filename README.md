# steal-a-brainrot
Steal a Brainrot est un jeu vidéo multijoueur sur Roblox, créé en 2025 et développé par SpyderSammy (également connu sous le nom de Sammy [1] ). Le jeu a été mis en ligne sur Roblox le 16 mai 2025. Dans ce jeu, les joueurs achètent et volent des Brainrots, des personnages inspirés du mème italien Brainrot,.
